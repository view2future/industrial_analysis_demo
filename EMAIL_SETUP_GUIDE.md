# 163邮箱IMAP配置指南

要使政策分析系统能够从163邮箱读取邮件，您需要完成以下设置：

## 1. 启用IMAP/SMTP服务

1. 登录163邮箱网页版：https://mail.163.com
2. 点击右上角的"设置"图标
3. 选择"POP3/SMTP/IMAP"
4. 找到"IMAP"服务，点击"开启"
5. 按照提示完成验证（通常需要手机验证）

## 2. 生成授权码

1. 在"POP3/SMTP/IMAP"设置页面
2. 找到"客户端授权密码"或类似选项
3. 点击"开启"或"生成授权码"
4. 按照提示完成验证，获取授权码

## 3. 更新配置文件

将获取的授权码替换config.json中的password字段：

```json
{
  "email": {
    "server": "imap.163.com",
    "port": 993,
    "username": "18611313200@163.com",
    "password": "您的授权码",  // 使用授权码而非登录密码
    "folder": "INBOX"
  }
}
```

## 4. 常见问题

### 错误："Unsafe Login"
- 原因：未开启IMAP服务或使用了登录密码而非授权码
- 解决：确保已按上述步骤开启服务并使用授权码

### 连接超时
- 检查网络连接
- 确认服务器地址：imap.163.com
- 确认端口：993（SSL）

### 登录失败
- 确认用户名格式：完整邮箱地址
- 确认使用的是授权码而非登录密码
- 检查是否有输入错误

## 5. 测试连接

配置完成后，运行测试脚本验证连接：
```bash
python3 test_email_reader.py
```

如果看到"✅ 邮箱连接成功"，说明配置正确。
