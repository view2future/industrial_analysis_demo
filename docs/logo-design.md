# 首页核心功能Logo设计说明

## 设计目标
- 与现有UI风格完全统一（渐变、圆角、阴影、字体）
- 现代、简洁、高识别度，专业且一致
- 适配响应式与深浅两种主题

## 风格与规范
- 画布：128×128 px（视口），基础输出尺寸 128×128；支持 32/64/128/256
- 背景：径向冷色系渐变（浅蓝→中蓝），配合卡片白底保持对比度
- 前景：白色线稿（3px描边），圆角连接，具备轻微投影提升层次
- 一致性：统一圆角半径、线宽、阴影与构图留白

## 模块与图形语义
- AI智能报告：文档框与AI头部（机器人光点）
- 多文档对比：两份文档并列，左右箭头表示比对
- 趋势预测：折线走势与虚线预测段（上升趋势）
- 知识图谱：节点与连线构成简化网络结构
- 地图可视化：柱状图与地图定位针组合
- 产业链图谱：上下游链条（链环+连接线）
- 政策解读：文档与批注符号（十字标记表示重点）
- 术语词典：书本与放大镜，左上角“A”代表术语

## 资源与导出
- 源文件（SVG）：`static/img/logos/*.svg`
- PNG导出：运行 `tools/export_logos.py`（需要 `cairosvg`）
  - 安装：`pip install cairosvg`
  - 导出：`python3 tools/export_logos.py`
  - 输出目录：`static/img/logos/png/`（生成 32/64/128/256 四种尺寸）

## 应用与预览
- 首页替换：`templates/index_enhanced.html` 使用 `<img class="feature-logo" ...>`
- 预览页面：`/logo_preview`（`templates/logo_preview.html`）展示多尺寸效果

## 深浅主题适配
- 背景渐变自带对比度，前景线稿为纯白；在深色主题下与卡片/页面背景形成清晰对比
- 建议避免在纯白背景上直接置灰线稿；保持卡片阴影与内边距确保可读性

## 未来扩展
- 增加品牌主题变体（紫/橙倾向）
- 为超小尺寸（≤24px）提供实心简化形态
- 可将SVG导入AI进行进一步排版或印刷级输出
