# UI测试计划（跨浏览器与跨设备）

## 浏览器与设备
- 桌面：Chrome 最新、Edge 最新、Safari 最新（macOS）
- 移动：iOS Safari（最新两版）、Android Chrome（最新两版）
- 分辨率：320/375/768/1024/1280/1440/1920 px

## 测试项
- 布局与响应：导航/卡片/表格/弹窗在各断点不溢出且层级正确
- 表单交互：`hover/active/focus/disabled` 全覆盖；输入焦点发光与错误提示对比明确
- 图标与按钮：尺寸/颜色一致；点击区域≥40×40；键盘焦点可见
- 表格：斑马纹、悬停、选中、分页与徽章状态显示一致
- 进度与阶段：霓虹内发光显示稳定；动画无卡顿
- 可访问性：对比度 AA；键盘导航可达；`aria-label` 在图标按钮上生效

## 截图与对比
- 页面级：首页（Hero/功能卡）、报告生成、流式生成、报告查看
- 组件级：按钮（6态）、输入（4态）、表格（头/行/状态）、弹窗/提示（4类）
- 输出：改造前/后同区域截图，标注差异与说明

## 质量门槛
- 无严重布局错位与遮挡；交互态完整；无高风险控制台错误
- ESLint/Stylelint 无阻断错误，告警逐步清零
