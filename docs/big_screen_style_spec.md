# 成都实时产业监测大屏 · 视觉与交互规范（v1.0）

## 色彩规范
- 背景：深色科技风（深蓝/黑） `#0f172a`
- 面板：半透明深灰 `rgba(31,41,55,0.6)`
- 文本：主要 `#e5e7eb`，次要 `#9ca3af`
- 指示色：蓝 `#3b82f6`，青 `#22d3ee`，绿 `#22c55e`，橙 `#f59e0b`，红 `#ef4444`

## 字体与字号
- 字体族：Inter/SF Pro/系统默认
- 标题：12–14px，小写标题统一字重 600
- 正文：12px，次要信息 11px
- 数字标签：对比色并加粗，保持单位与小数统一

## 布局与间距
- 栅格：12列响应式；4K/2K/1080p 分别调整列宽与间距
- 间距：组件外间距 24px；内间距 16px
- 圆角：16px；阴影：0 20px 40px rgba(0,0,0,0.35)

## 组件样式
- 面板容器：半透明背景、柔和边框与阴影、轻微悬浮动效
- 仪表盘：三段阈值分色，刻度清晰，数值居中显示
- 列表卡片：信息层次“标题/要点/标签”，对齐统一，交互反馈轻微
- 图表：统一配色与字体；坐标轴与图例采用次要文本色

## 动画与交互
- 过渡：240ms ease；悬浮位移 2px
- 滚动播报：30s 循环；避免过快影响可读性
- 钻取：点击列表/地图标记展开详情；统一面包屑导航

## 主题模式
- 暗黑：默认；明亮：背景与文本按变量切换
- 通过 `.theme-dark/.theme-light` 切换 CSS 变量实现

## 可视化规范
- 单位与口径：每个图表展示单位与时间范围
- 颜色语义：绿(正常)/橙(预警)/红(异常)
- 标注：关键结论配数据与方法说明

## 性能与兼容
- 刷新策略：最小 1 分钟；数据分组增量刷新
- 浏览器：Chrome/Edge/Safari/Firefox 主流版本
- 优化：复用图表实例、减少重绘、控制动画频率

## 品牌一致性
- 统一图标与配色；平台标题、徽章与标签风格一致
